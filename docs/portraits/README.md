Drop character portraits here as PNG files.

Use one file per NPC, matching the profile IDs in `lib/npc-data.ts`:

- `tom.png`
- `ben.png`
- `celia.png`
- `henri.png`
- `rosie.png`
- `jeff.png`
- `antoine.png`
- `quentin.png`

The demo expects these files at `/portraits/<name>.png`, but it will fall back
to the character monogram if a portrait is missing.
