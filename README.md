# Volet 168H — Animal Talking

Application POC Next.js + TypeScript.

## Prérequis

- Node.js 20+
- Chrome ou Edge récent (WebGPU requis pour le LLM, issues suivantes)

## Démarrage

```bash
cd volet-168h
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
